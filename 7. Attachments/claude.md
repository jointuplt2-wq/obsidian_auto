# 7. Attachments

## 목적과 역할
이미지·PDF 등 **바이너리 첨부 파일**의 중앙 보관소. 마크다운과 분리해 관리합니다.

## 어떤 파일이 들어가는가
- 노트에 임베드되는 이미지(`![[그림.png]]`), 다이어그램, 스캔본, PDF 등
- 텍스트가 아닌 모든 첨부물

## 작업 방식과 규칙
- Obsidian 설정에서 첨부 파일 기본 저장 위치를 이 폴더로 지정한다.
- 파일명은 의미 있게(날짜/주제). 노트 이동 시 임베드 링크는 Obsidian이 자동 갱신.
- 대용량 파일은 Git LFS 사용을 검토하고, 미사용 첨부는 주기적으로 정리한다.
- 여기에는 마크다운 노트를 두지 않는다(이 `claude.md` 제외).
