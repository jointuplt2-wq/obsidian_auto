# 5. Zettelkasten

## 목적과 역할
시간에 구애받지 않는 **영구 지식의 발전소**입니다. PARA가 "작업대"라면 여기는 "지식 자산".
세 단계 흐름으로 운영합니다: **00. Inbox → 10. Literature → 20. Permanent**.

## 흐름
```
포착(빠른 메모)        → 00. Inbox/
원천 정리(저자의 생각) → 10. Literature/
재구성(나의 통찰)      → 20. Permanent/
```

## 작업 방식과 규칙
- **참조 우선 / 복사 금지**: 노트끼리는 `[[링크]]`로 연결, 원본은 단일 진실 공급원.
- **연결 강제**: 새 Permanent 노트는 최소 2개 이상의 기존 노트와 연결한다.
- **원자성**: 하나의 노트 = 하나의 아이디어.
- PARA(`1.`~`4.`)에서 지식이 필요하면 여기 노트를 **참조**한다. 여기에 프로젝트 작업물을 넣지 않는다.
- 각 하위 폴더의 `claude.md` 규칙을 따른다.
