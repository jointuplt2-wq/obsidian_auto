## 🏠 당신의 Obsidian = 집

집에 방이 여러 개 있는 것처럼, 폴더마다 역할이 달라요:

### 📁 PARA 시스템 = "할 일" 공간 (시간 있음 ⏰)
| 폴더 | 역할 | 비유 🏠 |
|------|------|---------|
| **1. Projects** | 목표+마감일 있는 일 | 지금 하는 숙제 📝 (예: "웹사이트 만들기") |
| **2. Areas** | 계속 관리하는 것 | 매일 돌봐야 할 것 🌱 (예: "건강", "영어 공부") |
| **3. Resources** | 관심 주제 자료 | 참고 서랍 🗄️ (예: "프로그래밍") |
| **4. Archive** | 끝난 것 보관 | 창고 📦 |

### 💎 Zettelkasten = "지식" 공간 (시간 없음 ♾️)
| 폴더 | 역할 |
|------|------|
| **00. Inbox** 📥 | 빠른 메모 (임시) |
| **10. Literature** 📚 | 저자의 생각 (어제 만든 것!) |
| **20. Permanent** 💎 | 내 생각 (어제 만든 삼각형!) |

### 🛠️ 도우미 폴더
- **0. Docs** 📄 — 규칙 설명서
- **6. Templates** 📋 — 노트 자동 양식
- **7. Attachments** 🖼️ — 이미지, PDF

---

## 💡 가장 중요한 차이 (한 문장!)
> **PARA = "무엇을 할까?" (일)** ⏰
> **Zettelkasten = "무엇을 알까?" (지식)** 💎

**예시로 당신의 계획을 넣어보면:**
- ✅ **To-do 리스트** → `1. Projects` (할 일!)
- 🌱 **매일 영어 공부** → `2. Areas` (계속 관리!)
- 📅 **데일리 노트** → `Daily notes` 기능 (매일 기록)
- 💎 **영어로 배운 지식** → `5. Zettelkasten` (지식!)
