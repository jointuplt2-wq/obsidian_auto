# 4. Archive

## 목적과 역할
완료되었거나 더 이상 활성화되지 않은 항목의 **냉동 보관소**입니다.
삭제하지 않고 보관하여 검색·재활성화가 가능하게 합니다.

## 어떤 노트가 들어가는가
- 완료된 `1. Projects/` 프로젝트
- 중단된 `2. Areas/` 영역
- 비활성 `3. Resources/` 자료
- 원래의 `type`은 유지하고 `status`만 `completed`/`archived`로 변경

## 작업 방식과 규칙
- 들어올 때 `status`와 `updated`를 갱신한다.
- **편집하지 않는다** — 보존이 원칙. 다시 필요하면 원래 폴더로 되살린다.
- `5. Zettelkasten/`의 영구 지식은 절대 여기로 오지 않는다(지식은 보관 대상이 아님).
